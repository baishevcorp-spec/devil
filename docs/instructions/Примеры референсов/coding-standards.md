# Стандарты кода

## Форматирование
- Отступы: 2 пробела
- Кавычки: одинарные 'string'
- Точки с запятой: обязательны
- Запятые: после последнего элемента в объектах/массивах
- Максимальная длина строки: 100 символов

## TypeScript
- Строгая типизация (strict: true)
- Избегать `any`, использовать `unknown` или конкретные типы
- Использовать интерфейсы для публичных API
- Использовать дженерики для переиспользуемого кода

## React (если применимо)
- Функциональные компоненты с хуками
- Явные типы пропсов
- Избегать inline-стилей (использовать CSS-модули или styled-components)
- Мемоизация через useMemo/useCallback для тяжёлых вычислений

## Git
- Коммиты: conventional commits (feat:, fix:, docs:, refactor:)
- Ветки: feature/xxx, fix/xxx, hotfix/xxx
- PR: минимум 1 ревью перед мержем
