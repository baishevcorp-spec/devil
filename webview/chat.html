Функция activate в extension.ts

Функция activate — это точка входа расширения VS Code. Она вызывается автоматически, когда VS Code активирует расширение (например, при выполнении команды).

Что делает функция:

  1. Выводит сообщение в консоль: "Devil extension is now active!"
  2. Регистрирует команду devil.hello через vscode.commands.registerCommand
  3. Добавляет команду в context.subscriptions для автоматической очистки ресурсов

Пример кода:

typescript 
export function activate(context: vscode.ExtensionContext): void {
  console.log('Devil extension is now active!');

  const disposable = vscode.commands.registerCommand('devil.hello', () => {
    vscode.window.showInformationMessage('Devil: расширение работает!');
  });

  context.subscriptions.push(disposable);
}

Ключевые моменты:

  • context: vscode.ExtensionContext — объект контекста, предоставляемый VS Code
  • context.subscriptions — массив ресурсов, которые VS Code автоматически очистит при деактивации
  • vscode.commands.registerCommand — регистрирует команду, которую можно вызвать из палитры команд
