# Devil — AI Assistant for Developers

Интеллектуальный агент-ассистент для VS Code с долговременной памятью.

## Основная концепция

Devil — это расширение для Visual Studio Code, которое интегрирует возможности искусственного интеллекта в процесс разработки программного обеспечения. Ассистент использует OpenAI-совместимые API для генерации кода, ответов на вопросы и помощи в управлении проектами. Ключевое отличие Devil — графовая память, которая хранит контекст проекта и позволяет ассистенту «помнить» структуру кода и историю изменений.

## Ключевые возможности

- **Чат с LLM** — взаимодействие с искусственным интеллектом через OpenAI-совместимые API (GPT-4o-mini и другие)
- **Графовая память** — хранение информации о проекте в виде графа узлов (классы, функции, файлы) с возможностью поиска и навигации
- **Управление проектом** — сканирование структуры кода, построение зависимостей, индексация символов
- **Поиск по коду** — быстрый поиск совпадений в файлах с результатами в Markdown
- **Генерация кода** — создание кода, патчей и тестов на основе естественных запросов
- **Интеграция с Git** — просмотр diff, истории коммитов, изменений в файлах

## Основные сценарии использования

Devil предназначен для разработчиков, которые хотят ускорить процесс разработки за счёт автоматизации рутинных задач:

- **Анализ кода** — поиск функций, классов, интерфейсов по имени, анализ структуры проекта
- **Разработка** — генерация кода на основе описания, создание тестов, написание документации
- **Отладка** — поиск ошибок, анализ изменений, просмотр истории коммитов
- **Документирование** — создание и обновление документации по проекту вMarkdown

## Целевая аудитория

- Разработчики, работающие с большими кодовыми базами
- Команды, использующие TypeScript и Node.js
- Студенты и преподаватели, изучающие архитектуру VS Code-расширений
- Разработчики, заинтересованные в интеграции LLM в рабочий процесс
